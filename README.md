#### 软件架构
兼容性
- iPhone
设备需装有 iOS 13.0 或更高版本。
- iPad
设备需装有 iPadOS 13.0 或更高版本。
- iPod touch
设备需装有 iOS 13.0 或更高版本。
- Mac
需使用运行 macOS 11.0 或更高版本的系统并装有 Apple M1 芯片的 Mac。
